Hello, viewer of source! For application instructions, execute "hireMe()" in the JavaScript console.

Apply to Urban Airship's Web Team!

First, send us a request.

Send an HTTP POST request to /api/candidates.

Your first request will fail, but the response will tell you how to fix it.

Once you succeed, the response will include a unique identifier.

Next, write a cover letter.

Tell us why you'd like to work with us.

Include your unique ID and the source code or command-line invocation you used to send your POST request. That might make your cover letter long. That's okay!

Finally, submit your cover letter with your application.

Complete the application process wherever you found the posting that led you here.

We'll get back to you as soon as we can!
